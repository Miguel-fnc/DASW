¿Que es github y para que sirve?

Github es una plataforma en la nube en la que los desarrolladores puedes crear, compartir y mantener software

¿Comandos basicos?

- git init (inicializa el nuevo repositorio en el directorio actual)
- git clone (clona un repositorio de github en tu maquina local)
- git status (te muestta el estado del repositorio)
- git add (agrega un archivo en el area de preparacion para ser incluido en el proximo commit)
- git add (agrega todos los cambios)
- git commit (crea un commit con los cambios que han sido agregados)
- git push (envia los commits locales al repositorio)
- git log (muestra el historial de commits en el repositorio)
- git reset (deshace los cambios realizados en el area de preparacion de los commits)
